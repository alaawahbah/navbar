# Responsive navbar

Demo version: https://russmaxdesign.github.io/navbar/

A complex, responsive navbar that includes site utilites, a primary navigation and a secondary navigation that sits at the bottom of the screen for small screen devices.

The following should occur:

- At wide screen size, the full top bar should be displayed, with all items present.
-  At medium screen size, the current organisation name should be hidden within the top nav bar.
-  At small screen size, all primary items and the current organisation should be hidden within the top nav bar, and a new nav bar should be displayed, fixed to the bottom of the page.
